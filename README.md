# Sentiment Analysis for Twitter Data

## Project Description
This project focuses on the end-to-end pipeline of analyzing social media sentiments. It was developed by a team to process raw tweets, clean the data, and extract meaningful insights regarding public opinion on various entities.

## My Role & Technical Contribution
As a key contributor to this project, my work focused on the **Data Foundation and Preprocessing Phase**, which is critical for any research-based analysis. My tasks included:

* **Data Integrity & Diagnostics:** Developed the `summary()` function to detect missing values, duplicates, and data distribution using heatmaps.
* **Text Preprocessing Pipeline:** Designed the `clean_text()` and `clean_dataset()` functions. This involved:
    * Removing noise (URLs, @mentions, Special Characters).
    * Text Normalization (Lowercasing, stripping whitespaces).
    * Handling Null values and data deduplication.
* **Problem Formulation:** Translating the research problem into technical requirements within the class structure.

## Tools Used
* Python (Pandas, NumPy, Re)
* Visualization (Seaborn, Matplotlib)
* Jupyter Notebook
